Transform

transform: translate(10px,20px)
It is position on 10px in X-direction and 20px in Y direction from its parent

transform: translate(40deg)

It is rotated by 40deg clockwise

transform: scale(1,1)

It is scale to 1,1

transform: scale(2,3)

It is scale to 2,3

transform: skewX(20deg)

It is skew in X direction by 20deg

transform: skewY(-10deg)

It is skew in Y direction by -10deg (Default:Left Up to Right Down)

transform: skew(10deg,20deg)

It is skew in 10deg from X and 20deg from Y

matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())
matrix(1, -0.3, 0, 1, 0, 0)
